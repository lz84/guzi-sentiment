## 一、项目背景

为谷子量化交易系统提供舆情数据支撑，收集股票相关信息、国际局势、网络舆情等，辅助投资决策。

**核心场景**：
1. **A股舆情监测** - 中文财经新闻、股吧、微博
2. **Polymarket预测** - 英文政治/经济事件（Twitter、Reuters、BBC）
3. **国际局势分析** - 地缘政治、央行政策、重大事件

### 3.2 详细模块说明

#### 模块一：数据采集层

| 子模块 | 功能 | 数据源 | 更新频率 |
|--------|------|--------|----------|
| **新闻采集** | 财经新闻抓取 | 新浪财经、东方财富、财联社 | 实时 |

#### 模块二：数据处理层

| 子模块 | 功能 | 技术方案 |
|--------|------|----------|
| **文本清洗** | 去噪、格式化 | 正则表达式、BeautifulSoup |

| **实体识别** | 公司、人物、地点 | NER模型 |
| **主题提取** | 关键话题识别 | LDA主题模型、BERTopic |
| **事件抽取** | 重要事件识别 | 规则+模型混合 |
| **关系抽取** | 实体关系图谱 | 知识图谱技术 |

**情感分析模型建议**：
- 中文金融领域：FinBERT-Chinese
- 通用中文：RoBERTa-wwm-ext
- 英文金融：FinBERT

### 14.2 robots.txt 遵守策略

```python
# 示例：遵守robots.txt的爬虫配置
import robotexclusionrulesparser

def check_robots_txt(url, user_agent="SentimentBot"):
    parser = robotexclusionrulesparser.RobotExclusionRulesParser()
    parser.fetch(url)
    return parser.is_allowed(user_agent, url)
```
